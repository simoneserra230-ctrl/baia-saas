8. Sicurezza

BA.IA implementa misure tecniche e organizzative adeguate al rischio, tra cui:

  • Password hash con bcrypt (fattore di costo 12) — nessuna password è memorizzata in chiaro
  • Token di sessione randomici a 256 bit (crittograficamente sicuri)
  • Rate limiting su login, registrazione e reset password per prevenire attacchi brute-force
  • Validazione del tipo di file (magic bytes) per i documenti caricati
  • Log di audit per tutte le operazioni sensibili
  • Comunicazioni cifrate via HTTPS in produzione
  • CORS configurato per accettare richieste solo dai domini autorizzati
